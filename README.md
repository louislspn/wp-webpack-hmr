# wp-webpack-hmr

A blank WordPress theme with :

- Webpack bundler
- Browsersync
- Sass to CSS minifier
- Babel transpiler

## Features✨

- 1 configuration file to match your local server path
- 1 dev bundler
- 1 production bundler
- Auto-reload on file changes
